# Pixelated Portfolio Design Style Guide

## Design Philosophy
**Theme**: Dark Cyberpunk Pixel Art Aesthetic
**Mood**: Mysterious, Professional, Cutting-edge Retro
**Visual Language**: 8-bit sophistication meets modern web design

## Color Palette
**Primary Dark**: #0a0a0a (Deep Black)
**Secondary Dark**: #1a1a2e (Midnight Blue)
**Accent Purple**: #16213e (Deep Indigo)
**Neon Cyan**: #00d4ff (Electric Cyan)
**Neon Pink**: #ff006e (Magenta Pink)
**Neon Green**: #39ff14 (Cyber Green)
**Text Light**: #e0e0e0 (Soft White)
**Text Dim**: #888888 (Muted Gray)

## Typography
**Display Font**: "Orbitron" - Futuristic, geometric sans-serif for headings
**Body Font**: "JetBrains Mono" - Monospace font that complements pixel aesthetic
**Pixel Font**: "Press Start 2P" - Classic 8-bit style for special elements

## Visual Effects & Animation

### Background Effects
- **Animated Pixel Constellation**: Floating particles that create constellation patterns
- **Cyber Grid Overlay**: Subtle grid lines with animated pulse effects
- **Depth Parallax**: Multiple layers creating depth with pixel-perfect positioning

### Text Effects
- **Typewriter Animation**: Text appears character by character with pixel cursor
- **Neon Glow Pulse**: Headings with animated neon glow effects
- **Pixel Split Reveal**: Text reveals with pixelated transition effects
- **Color Cycling**: Accent text cycles through neon colors

### Interactive Elements
- **8-bit Hover States**: Buttons transform with pixel-perfect scaling
- **Particle Trail**: Mouse movement leaves temporary pixel trails
- **Glitch Effects**: Brief digital glitch animations on interactions
- **Scanline Overlay**: Subtle CRT monitor scanline effects

### Component Styling
- **Pixel Borders**: All containers have 2px pixel-perfect borders
- **Neon Shadows**: Box shadows with neon color tints
- **Gradient Overlays**: Subtle gradients from dark to accent colors
- **Animated Underlines**: Links with pixelated underline animations

## Layout Principles
- **Grid System**: 8px base unit for perfect pixel alignment
- **Asymmetric Balance**: Dynamic layouts with intentional imbalance
- **Depth Layering**: Multiple z-index layers for visual depth
- **Responsive Pixels**: Maintain pixel aesthetic across all screen sizes

## Animation Library Integration
- **Anime.js**: Smooth property animations and timelines
- **p5.js**: Canvas-based particle systems and generative art
- **PIXI.js**: WebGL-powered visual effects and shaders
- **Typed.js**: Typewriter text effects
- **Splitting.js**: Advanced text animation capabilities

## Interactive Components Style
- **Pixel Buttons**: 3D pixel art style with hover animations
- **Neon Form Fields**: Input fields with glowing borders and focus states
- **Animated Cards**: Project cards with 8-bit transition effects
- **Particle Interactions**: Click and hover effects spawn pixel particles

## Background Treatment
**Consistent Dark Theme**: Deep black (#0a0a0a) base with subtle texture overlays
**Animated Elements**: Floating pixel particles, grid lines, and constellation patterns
**No Section Backgrounds**: Content differentiated through spacing and visual elements
**Depth Through Layers**: Multiple animation layers create visual depth without color variation
